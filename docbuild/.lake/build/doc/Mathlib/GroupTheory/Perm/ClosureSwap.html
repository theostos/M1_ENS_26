    Subgroups generated by transpositions #

    This file studies subgroups generated by transpositions.

    Main results #

    • swap_mem_closure_isSwap : If a subgroup is generated by transpositions, then a transposition swap x y lies in the subgroup if and only if x lies in the same orbit as y.
    • mem_closure_isSwap : If a subgroup is generated by transpositions, then a permutation f lies in the subgroup if and only if f has finite support and f x always lies in the same orbit as x.
