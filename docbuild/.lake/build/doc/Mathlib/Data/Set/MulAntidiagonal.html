    def Set.mulAntidiagonal {α : Type u_1} [Mul α] (s t : Set α) (a : α) :
    Set (α × α)

    Set.mulAntidiagonal s t a is the set of all pairs of an element in s and an element in t that multiply to a.

    Equations
