    Partition of unity and convex sets #

    In this file we prove the following lemma, see exists_continuous_forall_mem_convex_of_local. Let X be a normal paracompact topological space (e.g., any extended metric space). Let E be a topological real vector space. Let t : X ‚Üí Set E be a family of convex sets. Suppose that for each point x : X, there exists a neighborhood U ‚àà ùìù X and a function g : X ‚Üí E that is continuous on U and sends each y ‚àà U to a point of t y. Then there exists a continuous map g : C(X, E) such that g x ‚àà t x for all x.

    We also formulate a useful corollary, see exists_continuous_forall_mem_convex_of_local_const, that assumes that local functions g are constants.
