    The basic continuous linear maps associated to ℂ #

    The continuous linear maps Complex.reCLM (real part), Complex.imCLM (imaginary part), Complex.conjCLE (conjugation), and Complex.ofRealCLM (inclusion of ℝ) were introduced in Analysis.Complex.Basic. This file contains a few calculations requiring more imports: the operator norm and (for Complex.conjCLE) the determinant.
