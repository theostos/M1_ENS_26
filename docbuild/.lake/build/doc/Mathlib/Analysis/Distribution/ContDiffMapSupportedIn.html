    Continuously differentiable functions supported in a given compact set #

    This file develops the basic theory of bundled n-times continuously differentiable functions with support contained in a given compact set.

    Given n : ‚Ñï‚àû and a compact subset K of a normed space E, we consider the type of bundled functions f : E ‚Üí F (where F is a normed vector space) such that:

    • f is n-times continuously differentiable: ContDiff ‚Ñù n f.
    • f vanishes outside of a compact set: EqOn f 0 K·∂ú.

    The main reason this exists as a bundled type is to be endowed with its natural locally convex topology (namely, uniform convergence of f and its derivatives up to order n). Taking the locally convex inductive limit of these as K varies yields the natural topology on test functions, used to define distributions. While most of distribution theory cares only about C^‚àû functions, we also want to endow the space of C^n test functions with its natural topology. Indeed, distributions of order less than n are precisely those which extend continuously to this larger space of test functions.

    Main definitions #

    • ContDiffMapSupportedIn E F n K: the type of bundled n-times continuously differentiable functions E ‚Üí F which vanish outside of K.
    • ContDiffMapSupportedIn.iteratedFDerivWithOrderLM: wrapper, as a ùïú-linear map, for iteratedFDeriv from ContDiffMapSupportedIn E F n K to ContDiffMapSupportedIn E (E [√ói]‚ÜíL[‚Ñù] F) k K.
    • ContDiffMapSupportedIn.iteratedFDerivLM: specialization of the above, giving a ùïú-linear map from ContDiffMapSupportedIn E F ‚ä§ K to ContDiffMapSupportedIn E (E [√ói]‚ÜíL[‚Ñù] F) ‚ä§ K.
    • ContDiffMapSupportedIn.topologicalSpace, ContDiffMapSupportedIn.uniformSpace: the topology and uniform structures on ùìì^{n}_{K}(E, F), given by uniform convergence of the functions and all their derivatives up to order n.

    Main statements #

    • ContDiffMapSupportedIn.isTopologicalAddGroup, ContDiffMapSupportedIn.continuousSMul and ContDiffMapSupportedIn.instLocallyConvexSpace: ùìì^{n}_{K}(E, F) is a locally convex topological vector space.

    Notation #

    In the Distributions scope, we introduce the following notations:

    • ùìì^{n}_{K}(E, F): the space of n-times continuously differentiable functions E ‚Üí F which vanish outside of K.
    • ùìì_{K}(E, F): the space of smooth (infinitely differentiable) functions E ‚Üí F which vanish outside of K, i.e. ùìì^{‚ä§}_{K}(E, F).
    • N[ùïú; F]_{K, n, i} (or simply N[ùïú]_{K, n, i}): the ùïú-seminorm on ùìì^{n}_{K}(E, F) given by the sup-norm of the i-th derivative.
    • N[ùïú; F]_{K, i} (or simply N[ùïú]_{K, i}): the ùïú-seminorm on ùìì_{K}(E, F) given by the sup-norm of the i-th derivative.

    Implementation details #

    • The technical choice of spelling EqOn f 0 K·∂ú in the definition, as opposed to tsupport f ‚äÜ K is to make rewriting f x to 0 easier when x ‚àâ K.
    • Since the most common case is by far the smooth case, we often reserve the "expected" name of a result/definition to this case, and add WithOrder to the declaration applying to any regularity.
    • In iteratedFDerivWithOrderLM, we define the i-th iterated differentiation operator as a map from ùìì^{n}_{K} to ùìì^{k}_{K} without imposing relations on n, k and i. Of course this is defined as 0 if k + i > n. This creates some verbosity as all of these variables are explicit, but it allows the most flexibility while avoiding DTT hell.
