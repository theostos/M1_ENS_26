    Translating linear orders to ℤ #

    In this file we implement the translation of a problem in any linearly ordered type to a problem in ℤ. This allows us to use the lia tactic to solve it.

    While the core algorithm of the order tactic is complete for the theory of linear orders in the signature (<, ≤), it becomes incomplete in the signature with lattice operations ⊓ and ⊔. With these operations, the problem becomes NP-hard, and the idea is to reuse a smart and efficient procedure, such as lia.
