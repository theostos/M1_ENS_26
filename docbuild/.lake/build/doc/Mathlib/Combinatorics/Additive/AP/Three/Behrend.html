    Behrend's bound on Roth numbers #

    This file proves Behrend's lower bound on Roth numbers. This says that we can find a subset of {1, ..., n} of size n / exp (O (sqrt (log n))) which does not contain arithmetic progressions of length 3.

    The idea is that the sphere (in the n-dimensional Euclidean space) doesn't contain arithmetic progressions (literally) because the corresponding ball is strictly convex. Thus we can take integer points on that sphere and map them onto â„• in a way that preserves arithmetic progressions (Behrend.map).

    • Behrend.sphere: The intersection of the Euclidean sphere with the positive integer quadrant. This is the set that we will map on â„•.
    • Behrend.map: Given a natural number d, Behrend.map d : â„•â¿ â†’ â„• reads off the coordinates as digits in base d.
    • Behrend.card_sphere_le_rothNumberNat: Implicit lower bound on Roth numbers in terms of Behrend.sphere.
    • Behrend.roth_lower_bound: Behrend's explicit lower bound on Roth numbers.

    Turning the sphere into 3AP-free set #

    We define Behrend.sphere, the intersection of the $L^2$ sphere with the positive quadrant of integer points. Because the $L^2$ closed ball is strictly convex, the $L^2$ sphere and Behrend.sphere are 3AP-free (threeAPFree_sphere). Then we can turn this set in Fin n â†’ â„• into a set in â„• using Behrend.map, which preserves ThreeAPFree because it is an additive monoid homomorphism.
