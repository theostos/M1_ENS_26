    Ruzsa's covering lemma #

    This file proves the Ruzsa covering lemma. This says that, for A, B finsets, we can cover A with at most #(A + B) / #B copies of B - B.

    source
    theorem Finset.ruzsa_covering_mul {G : Type u_1} [Group G] {K : ℝ} [DecidableEq G] {A B : Finset G} (hB : B.Nonempty) (hK : ↑(A * B).card ≤ K * ↑B.card) :
    ∃ F ⊆ A, ↑F.card ≤ K ∧ A ⊆ F * (B / B)
