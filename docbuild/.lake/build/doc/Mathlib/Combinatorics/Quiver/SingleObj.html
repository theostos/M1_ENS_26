    Single-object quiver #

    Single object quiver with a given arrows type.

    Main definitions #

    Given a type Œ±, SingleObj Œ± is the Unit type, whose single object is called star Œ±, with Quiver structure such that star Œ± ‚ü∂ star Œ± is the type Œ±. An element x : Œ± can be reinterpreted as an element of star Œ± ‚ü∂ star Œ± using toHom. More generally, a list of elements of a can be reinterpreted as a path from star Œ± to itself using pathEquivList.
