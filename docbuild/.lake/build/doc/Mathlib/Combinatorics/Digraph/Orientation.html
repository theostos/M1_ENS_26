    Graph Orientation #

    This module introduces conversion operations between Digraphs and SimpleGraphs, by forgetting the edge orientations of Digraph.

    Main Definitions #

    • Digraph.toSimpleGraphInclusive: Converts a Digraph to a SimpleGraph by creating an undirected edge if either orientation exists in the digraph.
    • Digraph.toSimpleGraphStrict: Converts a Digraph to a SimpleGraph by creating an undirected edge only if both orientations exist in the digraph.
