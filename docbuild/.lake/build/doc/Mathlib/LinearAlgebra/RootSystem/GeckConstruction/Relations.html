    Relations in Geck's construction of a Lie algebra associated to a root system #

    This file contains proofs that RootPairing.GeckConstruction.lieAlgebra contains sl₂ triples satisfying relations associated to the Cartan matrix of the input root system.

    Main definitions: #

    • RootPairing.GeckConstruction.isSl2Triple: a distinguished family of sl₂ triples contained in the Geck construction.
    • RootPairing.GeckConstruction.lie_h_e: an interaction relation between different sl₂ triples.
    • RootPairing.GeckConstruction.lie_h_f: an interaction relation between different sl₂ triples.
    • RootPairing.GeckConstruction.lie_e_f_ne: an interaction relation between different sl₂ triples.
