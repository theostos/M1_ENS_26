    Morphisms surjective on stalks #

    We define the class of morphisms between schemes that are surjective on stalks. We show that this class is stable under composition and base change.

    We also show that (AlgebraicGeometry.SurjectiveOnStalks.isEmbedding_pullback) if Y ‚ü∂ S is surjective on stalks, then for every X ‚ü∂ S, X √ó‚Çõ Y is a subset of X √ó Y (Cartesian product as topological spaces) with the induced topology.
