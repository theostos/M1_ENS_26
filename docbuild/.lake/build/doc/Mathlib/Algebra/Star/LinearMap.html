    Intrinsic star operation on E ‚Üí‚Çó[R] F #

    This file defines the star operation on linear maps: (star f) x = star (f (star x)). This corresponds to a map being star-preserving, i.e., a map is self-adjoint iff it is star-preserving.

    Implementation notes #

    Note that in the case of when E = F for a finite-dimensional Hilbert space, this star is mathematically distinct from the global instance on E ‚Üí‚Çó[ùïú] E where star := LinearMap.adjoint. For that reason, the intrinsic star operation is scoped to IntrinsicStar.
