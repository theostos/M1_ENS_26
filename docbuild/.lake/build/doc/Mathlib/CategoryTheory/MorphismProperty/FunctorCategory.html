    Stability properties of morphism properties on functor categories #

    Given W : MorphismProperty C and a category J, we study the stability properties of W.functorCategory J : MorphismProperty (J тед C).

    Under suitable assumptions, we also show that if monomorphisms in C are stable under transfinite compositions (or coproducts), then the same holds in the category J тед C.
