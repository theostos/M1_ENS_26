    Fibers of functors #

    In this file we introduce a typeclass HasFibers for a functor p : ğ’³ â¥¤ ğ’®, consisting of:

    We also provide a canonical HasFibers instance, which uses the standard fibers Fiber p S (see Mathlib/CategoryTheory/FiberedCategory/Fiber.lean). This makes it so that any result proven about HasFibers can be used for the standard fibers as well.

    The reason for introducing this typeclass is that in practice, when working with (pre)fibered categories one often already has a collection of categories Fib S for every S that are equivalent to the fibers Fiber p S. One would then like to use these categories Fib S directly, instead of working through this equivalence of categories. By developing an API for the HasFibers typeclass, this will be possible.

The associated fibered category then has objects (S, a) where S : ğ’® and a is an element of F(S). In this case, the HasFibers instance is given by the categories F(S) and the functor Î¹ sends a : F(S) to (S, a) in the fibered category.
