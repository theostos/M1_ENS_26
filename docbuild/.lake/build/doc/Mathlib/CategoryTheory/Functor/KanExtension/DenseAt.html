    Canonical colimits, or functors that are dense at an object #

    Given a functor F : C ‚•§ D and Y : D, we say that F is dense at Y (F.DenseAt Y), if Y identifies to the colimit of all F.obj X for X : C and f : F.obj X ‚ü∂ Y, i.e. Y identifies to the colimit of the obvious functor CostructuredArrow F Y ‚•§ D. In some references, it is also said that Y is a canonical colimit relatively to F. While F.DenseAt Y contains data, we also introduce the corresponding property isDenseAt F of objects of D.
