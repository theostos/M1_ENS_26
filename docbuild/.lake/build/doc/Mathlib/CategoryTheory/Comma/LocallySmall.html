    Comma categories are locally small #

    We introduce instances showing that the various comma categories are locally small when the relevant categories that are involved are locally small.

    source
    instance CategoryTheory.Comma.locallySmall {A : Type u₁} {B : Type u₂} {T : Type u₃} [Category.{v₁, u₁} A] [Category.{v₂, u₂} B] [Category.{v₃, u₃} T] (L : Functor A T) (R : Functor B T) [LocallySmall.{w, v₁, u₁} A] [LocallySmall.{w, v₂, u₂} B] :
