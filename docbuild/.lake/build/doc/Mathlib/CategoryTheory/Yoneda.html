    The Yoneda embedding #

    Let C : Type u be a category (with Category.{v} C). We define the Yoneda embedding as a fully faithful functor yoneda : C C Type v, In addition to yoneda, we also define uliftYoneda : C C Type (max w v) with the additional universe parameter w. When C is locally w-small, one may also use shrinkYoneda.{w} : C C Type w from the file CategoryTheory.ShrinkYoneda.

    The naturality of the bijection yonedaEquiv involved in the Yoneda lemma is also expressed as a natural isomorphism yonedaLemma : yonedaPairing C yonedaEvaluation C.
