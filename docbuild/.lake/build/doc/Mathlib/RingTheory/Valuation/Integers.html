      Prop

      Given a valuation v : R → Γ₀ and a ring homomorphism O →+* R, we say that O is the integers of v if f is injective, and its range is exactly v.integer.

      • exists_of_le_one ⦃r : R⦄ : v r ≤ 1 → ∃ (x : O), (algebraMap O R) x = r
