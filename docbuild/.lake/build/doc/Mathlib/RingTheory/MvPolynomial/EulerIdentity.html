    Euler's homogeneous identity #

    Main results #

    • IsHomogeneous.sum_X_mul_pderiv: Euler's identity for homogeneous polynomials: for a multivariate homogeneous polynomial, the product of each variable with the derivative with respect to that variable sums up to the degree times the polynomial.
    • IsWeightedHomogeneous.sum_weight_X_mul_pderiv: the weighted version of Euler's identity.
