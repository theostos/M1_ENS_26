    The de Rham Period Ring $\mathbb{B}_{dR}^+$ and $\mathbb{B}_{dR}$ #

    In this file, we define the de Rham period ring $\mathbb{B}_{dR}^+$ and the de Rham ring $\mathbb{B}_{dR}$. We define a generalized version of these period rings following Scholze.
