    def HahnSeries.SummableFamily.binomialFamily {Γ : Type u_1} {R : Type u_2} {A : Type u_3} [LinearOrder Γ] [AddCommMonoid Γ] [IsOrderedCancelAddMonoid Γ] [CommRing R] [BinomialRing R] [CommRing A] [Algebra R A] (x : HahnSeries Γ A) (r : R) :
    SummableFamily Γ A ℕ

    A summable family of Hahn series, whose nth term is Ring.choose r n • (x - 1) ^ n when x is close to 1 (more precisely, when 0 < (x - 1).orderTop), and 0 ^ n otherwise. These terms give a formal expansion of x ^ r as (1 + (x - 1)) ^ r.
