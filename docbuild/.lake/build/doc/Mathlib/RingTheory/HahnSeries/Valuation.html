    Valuations on Hahn Series rings #

    If Γ is a LinearOrderedCancelAddCommMonoid and R is a domain, then the domain R⟦Γ⟧ admits an additive valuation given by orderTop.

    Main Definitions #

    • HahnSeries.addVal Γ R defines an AddValuation on R⟦Γ⟧ when Γ is linearly ordered.
