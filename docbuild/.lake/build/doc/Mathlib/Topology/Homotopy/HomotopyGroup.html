    nth homotopy group #

    We define the nth homotopy group at x : X, œÄ_n X x, as the equivalence classes of functions from the n-dimensional cube to the topological space X that send the boundary to the base point x, up to homotopic equivalence. Note that such functions are generalized loops GenLoop (Fin n) x; in particular GenLoop (Fin 1) x ‚âÉ Path x x.

    We show that œÄ_0 X x is equivalent to the path-connected components, and that œÄ_1 X x is equivalent to the fundamental group at x. We provide a group instance using path composition and show commutativity when n > 1.
