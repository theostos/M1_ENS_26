    Proper group action #

    In this file we define proper action of a group on a topological space, and we prove that in this case the quotient space is T2. We also give equivalent definitions of proper action using ultrafilters and show the transfer of proper action to a closed subgroup.

    Main definitions #

    • ProperSMul : a group G acts properly on a topological space X if the map (g, x) ‚Ü¶ (g ‚Ä¢ x, x) is proper, in the sense of IsProperMap.

    Main statements #

    • t2Space_quotient_mulAction_of_properSMul: If a group G acts properly on a topological space X, then the quotient space is Hausdorff (T2).
    • t2Space_of_properSMul_of_t1Group: If a T1 group acts properly on a topological space, then this topological space is T2.

    References #

    • [N. Bourbaki, General Topology][bourbaki1966]
