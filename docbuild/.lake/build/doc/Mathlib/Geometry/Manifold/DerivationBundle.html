    Derivation bundle #

    In this file we define the derivations at a point of a manifold on the algebra of smooth functions. Moreover, we define the differential of a function in terms of derivations.

    The content of this file is not meant to be regarded as an alternative definition to the current tangent bundle but rather as a purely algebraic theory that provides a purely algebraic definition of the Lie algebra for a Lie group. This theory coincides with the usual tangent bundle in the case of finite-dimensional C^âˆ real manifolds, but not in the general case.
