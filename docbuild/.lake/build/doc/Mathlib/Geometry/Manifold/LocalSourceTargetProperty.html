    Local properties of smooth functions which depend on both the source and target #

    In this file, we consider local properties of functions between manifolds, which depend on both the source and the target: more precisely, properties P of functions f : M â†’ N such that f has property P if and only if there is a suitable pair of charts on M and N, respectively, such that f read in these charts has a particular form.
